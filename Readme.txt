React-Native

This is a project for Udacitiy. using react-native to build mobile application.

it is a mobile flash card game, where you can create you own decks and add cards to them also you can delete.
You can quiz your friends where they can chose either correct or false. You will get scored at the end. 

it was tested on Andriod phone.

all you have to do is follow these steps:- 
1- download the code from github.
2-after downloading open it with Visual Stdio Code.
3-in the terminal do npm install or yarn install.
4- also in the terminal do expo start.
5- it will open the browser you will find a QR code.
6- you must download the EXPO app from your play store or app store.
7-scan the QR code.

that is it enjoy.
